SKY-Vivaldi PC Web

Name

SKY-Vivaldi

Category

PC webpage

Project Date

Jun 29, 2020

Used Language

HTML : SASS : VanillaJS

Project Link

이 프로젝트는 우리나라 Sky-Vivaldi(스카이 비발디) 펜션의 pc 페이지를 리뉴얼 하였으며, 홍천 대명 비발디 입구에 위치하고 회사 세미나, 연인 숙박, 혼자 여행 등 편안한 휴식을 제공하는 펜션입니다. 메인 페이지부터 복잡한 사진들로 가득 차 유저들의 시선을 사로잡지 못하는 경향이 있습니다. 브랜드 컨셉 표현 또한 확고하지 못해 직접 시안을 잡고 작업을 진행했습니다.

콘텐츠는 각각 4개의 스크롤 형식으로 구성해 놓았으며 펜션만의 고객호실, 공지사항, 주변 관광지, 서비스들을 배치하였습니다. 해당하는 Header, Section, Footer 란마다 코딩의 상세 요약은 아래에 명시되어 있습니다.

Scroll Down

  • Header

    메뉴 및 슬라이더
    메뉴는 2depth형식으로 이루어져있으며 전체창이 슬라이드 다운되는 효과.
    이 프로젝트의 슬라이더는 하드코딩으로 진행하였으며,
    페이지네이션에 애니메이션효과를 투자하고 JS로 재생/정지버튼 구현
    슬라이더 밑 제휴처 로고들은 타임인터벌을 주어 자동으로 무빙되는 효과.
